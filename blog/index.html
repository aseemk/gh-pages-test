This little repo shows GitHub's cool URL rewriting with Pages.

This file is blog/index.html, and it can be accessed at:

  • /blog (redirects to next one)
  • /blog/
  • /blog/index.html
  • /blog/index (extension-less — isn't that cool?)

Now check out:

  • /blog/post1
  • /blog/post2
